# JNSQ Resonant Orbit Calculator

A web-based resonant orbit calculator for the [JNSQ](https://github.com/Galileo88/JNSQ) planet pack for Kerbal Space Program.

Based on [Eric Meyer's resonant orbit calculator](https://meyerweb.com/eric/ksp/resonant-orbits/) (MIT), adapted for JNSQ's 31 bodies.

## Usage

Open `index.html` in a browser. Select a body, enter your target altitude and satellite count, then read off the resonant orbit parameters and Δv.
